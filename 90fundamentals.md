  <h1>Vertica Fundamentals</h1>
  <p>Vertica 데이터베이스의 기본 개념부터 아키텍처, 운영, 관리에 이르는 28가지 핵심 학습 주제를 소개합니다. 각 주제는 향후 상세 콘텐츠로 업데이트될 예정입니다.</p>

  <div class="cards">
    <article class="card"><h3>1. Vertica 아키텍처</h3><p>MPP, Columnar, Projection 등 핵심 아키텍처를 이해합니다.</p></article>
    <article class="card"><h3>2. DB 기동 및 정지</h3><p>데이터베이스를 시작하고 중지하는 기본 명령어를 학습합니다.</p></article>
    <article class="card"><h3>3. Node 기동 및 정지</h3><p>클러스터 내 개별 노드를 관리하는 방법을 알아봅니다.</p></article>
    <article class="card"><h3>4. License</h3><p>Vertica 라이선스 정책과 사용량 확인 방법을 학습합니다.</p></article>
    <article class="card"><h3>5. User 관리</h3><p>사용자 생성, 수정, 삭제 등 계정 관리 방법을 알아봅니다.</p></article>
    <article class="card"><h3>6. Profile 관리</h3><p>사용자 세션의 리소스 및 암호 정책을 제어하는 프로필을 학습합니다.</p></article>
    <article class="card"><h3>7. Schema 관리</h3><p>객체를 논리적으로 그룹화하는 스키마의 개념과 활용법을 익힙니다.</p></article>
    <article class="card"><h3>8. Resource Pool</h3><p>워크로드별 자원을 격리하고 제어하는 리소스 풀을 알아봅니다.</p></article>
    <article class="card"><h3>9. Table 관리</h3><p>데이터 구조를 정의하는 테이블의 생성 및 관리 방법을 학습합니다.</p></article>
    <article class="card"><h3>10. Projection 관리</h3><p>쿼리 성능을 좌우하는 물리 객체인 프로젝션의 설계 방법을 익힙니다.</p></article>
    <article class="card"><h3>11. Privilege 관리</h3><p>객체 접근을 제어하는 권한(GRANT/REVOKE) 체계를 이해합니다.</p></article>
    <article class="card"><h3>12. 통계 정보 수집</h3><p>쿼리 최적화를 위한 통계(Statistics) 수집 및 관리 방법을 알아봅니다.</p></article>
    <article class="card"><h3>13. 세션 관리</h3><p>활성 세션을 모니터링하고 제어하는 방법을 학습합니다.</p></article>
    <article class="card"><h3>14. Purge 작업</h3><p>삭제된 데이터를 물리적으로 정리하는 Purge 매커니즘을 이해합니다.</p></article>
    <article class="card"><h3>15. 백업 (Backup)</h3><p>vbr 유틸리티를 이용한 전체/증분 백업 방법을 학습합니다.</p></article>
    <article class="card"><h3>16. 복구 (Restore)</h3><p>백업된 데이터를 사용하여 데이터베이스를 복구하는 절차를 익힙니다.</p></article>
    <article class="card"><h3>17. Database Designer</h3><p>프로젝션 설계를 자동화하는 Database Designer(DBD) 활용법을 알아봅니다.</p></article>
    <article class="card"><h3>18. 시스템 리소스 모니터링</h3><p>CPU, 메모리 등 시스템 자원 사용량을 모니터링하는 방법을 학습합니다.</p></article>
    <article class="card"><h3>19. Lock 모니터링</h3><p>데이터 경합을 유발하는 Lock 상태를 확인하고 해결하는 방법을 알아봅니다.</p></article>
    <article class="card"><h3>20. 성능 모니터링</h3><p>느린 쿼리, 리소스 과다 사용 쿼리 등 성능 병목을 찾는 방법을 익힙니다.</p></article>
    <article class="card"><h3>21. 데이터 분산 모니터링</h3><p>데이터가 노드 간에 균일하게 분산되었는지 확인하는 방법을 학습합니다.</p></article>
    <article class="card"><h3>22. 데이터 적재 (COPY)</h3><p>대용량 데이터를 빠르고 효율적으로 적재하는 COPY 문법을 익힙니다.</p></article>
    <article class="card"><h3>23. 데이터 Export</h3><p>vsql 또는 EXPORT 함수를 사용하여 데이터를 외부로 내보내는 방법을 알아봅니다.</p></article>
    <article class="card"><h3>24. Epoch 이해하기</h3><p>데이터 일관성을 보장하는 Epoch(LGE, AHM)의 개념을 이해합니다.</p></article>
    <article class="card"><h3>25. Tuple Mover</h3><p>WOS의 데이터를 ROS로 옮기는 Tuple Mover의 동작 원리를 학습합니다.</p></article>
    <article class="card"><h3>26. Node Down 장애 처리</h3><p>노드 장애 발생 시 복구 절차와 데이터 정합성 유지 원리를 알아봅니다.</p></article>
    <article class="card"><h3>27. System Hang 장애 처리</h3><p>시스템 행(Hang) 발생 시 원인 분석 및 해결 방법을 학습합니다.</p></article>
    <article class="card"><h3>28. 클라이언트 도구 활용</h3><p>vsql, DBeaver 등 다양한 클라이언트 도구의 사용법을 익힙니다.</p></article>
