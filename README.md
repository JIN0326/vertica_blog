# Vertica Blog

이 저장소는 `Vertica_소개자료_202509.pptx` 기반으로 구축된 GitHub Pages 블로그입니다.

## 현재 상태

- 새로운 `playbook.md` 페이지를 추가했습니다.
- `administration.md`를 `버티카_운영자가이드.doc` 기반 운영자 가이드 요약 페이지로 업데이트했습니다.
- `fundamentals.md`는 기존 구조를 유지하되 `index.md`와 스타일을 통일하도록 색상 및 카드 레이아웃을 정리했습니다.
- `index.md` 상단 네비게이션과 `Vertica Playbook` 섹션을 업데이트했습니다.
- `_layouts/default.html` 내 탐색 링크를 `playbook` 페이지로 연결되도록 수정했습니다.


## 사용 방법

1. 이 저장소를 GitHub Pages에 배포하려면 `main` 또는 `master` 브랜치의 루트 디렉터리를 GitHub Pages 소스로 지정합니다.
2. `_config.yml`을 필요에 따라 수정합니다.
3. 로컬에서 테스트하려면 Jekyll이나 다른 정적 사이트 서버를 사용할 수 있습니다.
