# Vertica Blog

이 저장소는 `Vertica_소개자료_202509.pptx`의 내용을 바탕으로 만든 GitHub Pages 블로그입니다.

## 사용 방법

1. GitHub 저장소를 생성하고 이 폴더를 푸시합니다.
2. GitHub Pages 설정에서 브랜치를 `main` 또는 `master`로 지정하고 루트 폴더를 선택합니다.
3. `_config.yml`에 필요한 설정을 수정합니다.

## 파일 구성

- `_config.yml`: GitHub Pages 설정
- `index.md`: 블로그 홈
- `what-is-vertica.md`: Vertica 개요
- `technology.md`: Vertica 핵심 기술
- `portfolio.md`: Vertica 포트폴리오
- `architecture.md`: 아키텍처 장점
- `operation.md`: 운영/관리 및 개발 편의성
